    Personas

    Introdução

      Uma persona é um personagem fictício, arquétipo hipotético de um grupo de usuários reais, criada para descrever um usuário típico (Cooper et al., 2007; Pruitt e Adlin, 2006; Cooper, 1999).

      Personas são utilizadas principalmente para representar um grupo de usuários finais durante discussões de design, mantendo todos focados no mesmo alvo.

      Vale ressaltar que para fazer as personas, utilizamos foto do site ThisPersonDoesNotExist, que gera imagens de pessoas que não existem, ou seja, nenhuma das imagens infringe alguma lei.

      Após a identificação desses grupos e de suas características predominantes, foi utilizado o site uxpressia para inspiração ao criar os cartões de cada persona.

      Os tipos de personalidade foram baseados no site vittude utilizando a técnica de eneagrama.
